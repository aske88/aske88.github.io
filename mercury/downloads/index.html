Downloads

Mercury Editor comes as a Rails Engine, but if you'd like to use Mercury outside of a typical Rails application, you can download these packages and use them.

Mercury expects a back end for many of its features, so you may have to duplicate some of that functionality in your own application. For information on how to do that you'll want to read the Using Mercury without Rails and Server Specification Wiki pages.
